# Sales Performance Dashboard (Power BI)

## Project Overview
This project is a Sales Performance Dashboard created using Power BI.
The dataset was cleaned using Python and visualized in Power BI to generate business insights.

## KPIs
- Total Sales
- Total Profit
- Total Orders
- Profit Margin

## Dashboard Features
- Sales by Category
- Sales by Segment
- Monthly Sales Trend
- Top 10 Products
- Region, Category, Segment Filters

## Tools Used
- Python (Pandas)
- Power BI
- Data Visualization
- DAX

## Insights
- Technology category has highest sales
- High discounts reduce profit
- West region generates highest revenue
